---

## Installation

```bash
npm install @mumbo-group/ui
```

---

## Usage

Here's a simple example using the common `Button` component in your React app:

```tsx
import React from "react";
import { Button } from "@mumbo-group/ui";

function App() {
  return (
    <div>
      <h1>UI Demo</h1>
      <Button>Hello</Button>
    </div>
  );
}

export default App;
```
